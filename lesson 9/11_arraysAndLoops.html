Array is an another type of value . It represents a list of values. Syntax : [val1, val2, ...]. Arrays can be written on multiple lines. We can access array values using square brackets with position specified in it ie index.

Arrays are references. ie variable stores reference only actual array is stored somewhere else in memory.

Loops let us run some code over and over. In jS 2 types of loops. 1) While loop 2) For loop. Use for loop for standard loops (loops with loop variable, condition and increment step) and use while loop for non-standard loops

Loops create a scope. Any variable created inside loop's curly braces are accessible only inside loop curly braces.
